# GameCritic Hub

## Description

GameCritic Hub is a web application where users can discover video games, write reviews, rate games, and maintain a personal watchlist of games they want to play. The platform is designed to help gamers share their opinions and find useful information about different games before trying them.

**Creating an Account**
A new user visits the website and decides to create an account. After registering and logging in, they can browse games, write reviews, and add games to their watchlist.

**Reviewing a Game**

A user opens the page of a video game they have played and decides to share their experience. They leave a rating and write a short review that other users can read when visiting the same game page.

**Administrator Moderating the Platform**

The administrator logs into the admin panel to manage the platform. They can add or edit games, delete inappropriate comments, and punish users who break the rules in order to keep the community respectful and organized.
